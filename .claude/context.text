== CONVENTIONS ==

Feature file:     features/<feature-name>.md
Test file:        tests/test-<feature-name>.js
Code file:        src/<feature-name>.js

Testing:          Vitest (JavaScript)
Pattern:          Tests verify requirements → Code passes tests
